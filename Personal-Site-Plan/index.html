Dynamic Elements (JavaScript)

JavaScript will be used to make the site more interactive and helpful. These are the features I plan to add:

  • Project Filter: I will use JavaScript to let users filter projects by type or technology. This will update the content on the page without reloading.
  • Form Validation: The contact form will check input in real-time. Users will get messages if they forget something or type it wrong.
  • Theme Toggle: Visitors can switch between dark and light mode. JavaScript will save their preference using local storage.
  • Animated Skill Bars: My skills section will use JavaScript to show progress bars that animate based on my current skill level.
  • Modular Code: I will organize JavaScript into separate files using ES modules. This makes it easier to manage and reuse code.
